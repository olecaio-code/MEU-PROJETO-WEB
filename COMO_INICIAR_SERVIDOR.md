# ⚠️ SERVIDOR NÃO ESTÁ RODANDO!

## ❌ Você recebeu: `ERR_CONNECTION_REFUSED`

Isso significa que **o servidor HTTP precisa ser iniciado** para servir os arquivos.

---

## ✅ COMO INICIAR O SERVIDOR

### **OPÇÃO 1: Duplo-clique (MAIS FÁCIL) ⭐**

1. Procure pelo arquivo: **`INICIAR_SERVIDOR.bat`**
2. **Duplo-clique** nele
3. Uma janela preta vai abrir
4. Aparecerá a mensagem:
   ```
   🚀 Servidor iniciado em http://localhost:8000
   ```

### **OPÇÃO 2: Prompt de Comando**

1. Abra **Prompt de Comando** (cmd.exe)
2. Digite:
   ```
   cd "c:\Users\Caio Bruno Corrêa\OneDrive\Área de Trabalho\meu-projeto-web"
   python -m http.server 8000
   ```
3. Verá:
   ```
   Serving HTTP on 0.0.0.0 port 8000
   ```

### **OPÇÃO 3: Node.js (se instalado)**

```
node simple-server.js
```
